# 💼 Employee Payroll Management System

A modern **Employee Payroll Management System** built using **Java (OOP)** with a **stylish dashboard UI (Swing)**.
This project demonstrates real-world use of **Object-Oriented Programming concepts** like inheritance, polymorphism, and abstraction.

---

## 🚀 Features

* 👨‍💼 Add employee details
* 💰 Automatic salary calculation
* 🔄 Supports multiple employee types

  * Full-Time Employees
  * Part-Time Employees
* 🧠 OOP-based architecture
* 🎨 Modern **Dark Dashboard UI (Apple-inspired)**
* ⚡ Fast and user-friendly

---

## 🧠 OOP Concepts Used

* **Inheritance** → `Employee → FullTimeEmployee / PartTimeEmployee`
* **Polymorphism** → Same method, different behavior
* **Abstraction** → Abstract `Employee` class
* **Method Overriding** → `calculateSalary()`
* **Encapsulation** → Data hiding inside classes

## 🧮 Salary Logic

* **Full-Time Employee**

```
Salary = Base Salary + Bonus
```

* **Part-Time Employee**

```
Salary = Hours Worked × Hourly Rate
```
